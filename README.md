# vue-seed

## Introduce
这是一个用`vue-cli`创建的Demo项目。

### Main feature
- Vuex
- Router
- Three.js
- Axios
- Vant
- Typescript

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
